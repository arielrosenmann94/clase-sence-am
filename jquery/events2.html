Eventos jQuery con animaciones (sin almacenar logs)

Cada evento dispara una animación visible (ideal para clases). No se guarda historial, solo se “ve” el efecto.

Mouse Click / dblclick Teclado Form (change/submit/focus) Load Scroll

1) mouseenter / mouseleave: tarjeta que se da vuelta

Cuando el mouse entra, la tarjeta rota (flip). Cuando sale, vuelve al frente.

Frente

Pasa el mouse encima para ver el reverso.

Reverso

mouseenter / mouseleave activan esta animación.

2) click y dblclick: efectos “pulse” y “shake”

Un click hace “pulse”. Un doble click hace “shake”.

Este bloque recibe las animaciones de pulse y shake.

3) keydown / keyup: “barra de energía”

Mantén presionada una tecla para subir la barra. Al soltar, la barra baja.

keydown aumenta; keyup reduce.

4) change / submit / focus / blur

Cambia el select para modificar el “tema”. En focus/blur se enciende/apaga un brillo. En submit aparece “enviado”.

Enviado

5) load: imagen pasa de borrosa a nítida

Cuando la imagen termina de cargar, se le quita el blur y se hace nítida.

6) scroll: barra de progreso superior

Al hacer scroll, la barra superior muestra el avance en la página.
