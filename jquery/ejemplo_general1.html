```html Animaciones jQuery para clases: show/hide, fade, slide, animate + menú

Sitio para clases: Animaciones de entrada/salida con jQuery

Incluye: show/hide/toggle, fadeIn/fadeOut/fadeToggle, slideDown/slideUp/slideToggle y animaciones personalizadas con animate().

show/hide fade slide animate() entradas/salidas extra resumen

1) show() / hide() / toggle()

show() muestra (quita display:none), hide() oculta (pone display:none), y toggle() alterna entre ambos estados.

Uso típico: mostrar/ocultar bloques rápido, sin transición.

SHOW/HIDE
Bloque visible/oculto sin animación.

    


    
    

      
2) fadeIn() / fadeOut() / fadeToggle()

      

        Cambian la opacidad gradualmente. Puedes indicar duración en ms (ej: 400).
      

      
Uso típico: mensajes, avisos, contenido que aparece “suave”.


      

        
        
        
      


      

        FADE
        
Bloque con transición de opacidad.

      


      

    


    
    

      
3) slideDown() / slideUp() / slideToggle()

      

        Animación vertical tipo acordeón: despliega y pliega ajustando la altura.
      

      
Uso típico: FAQ, acordeones, paneles de configuración.


      

        
        
        
      


      

        SLIDE
        

          Ejemplo: panel que se despliega/plega como acordeón.
          

        

      


      

    


    
    

      
4) animate(): animaciones personalizadas

      

        animate() permite animar propiedades numéricas CSS (por ejemplo: opacity,
        marginLeft, width, height). No anima todo (por ejemplo, no anima
        colores sin plugins).
      

      

        Uso típico: “entradas/salidas” con desplazamiento, resaltado temporal, expansión de panel, microinteracciones.
      


      

        
        
        
      


      

        ANIMATE
        

          Este bloque usa animate() para simular entrada/salida con desplazamiento y opacidad.
        

      


      

    


    
    

      
5) Otras entradas/salidas “para mostrar en clase”

      

        Aquí tienes combinaciones típicas: “pop” (escala+opacidad), “shake” (error), y “highlight” (resaltar).
        Todas hechas con jQuery (sin librerías extra).
      


      
      

        
A) Pop In / Pop Out (escala + opacidad)

        

          
          
        


        

          POP
          
Entrada/salida tipo “pop” (simula zoom suave).

        


        

      


      


      
      

        
B) Shake (para errores o validaciones)

        

          
        


        

          SHAKE
          
Haz clic en “Shake” para ver un efecto de error.

        


        

      


      


      
      

        
C) Highlight (resaltar temporalmente un bloque)

        

          
        


        

          HIGHLIGHT
          
Resalta el bloque para llamar la atención del usuario.

        


        

      

    


    
    

      
Resumen para explicar en clase

      
    
        
  • toggle / fadeToggle / slideToggle: alternan el estado sin if/else manual.
    • 
        
  • fade: opacidad (entradas/salidas suaves).
    • 
        
  • slide: altura (acordeones / FAQ).
    • 
        
  • animate: control total sobre propiedades numéricas (microinteracciones).
```
